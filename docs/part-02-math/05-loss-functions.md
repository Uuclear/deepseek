# 损失函数

> **前置知识**：熵、梯度  
> **预计时间**：50 分钟  
> **本章产出**：理解 MSE 与交叉熵

**MSE**：回归预测与真值差的平方均值。

**交叉熵**：分类中衡量预测分布与真实分布差异。

训练目标：最小化损失；优化器用梯度更新参数。
